Live Firebase Throughput Receiver

Iperfer Stream Plotter

This dashboard listens to the Firebase Realtime Database branch iperf3_streams. As soon as a client-side iperfer agent pushes throughput samples with an interface code, the website creates a live plot for that stream.

POST endpoint

Every client measurement agent pushes samples here. The interface being measured does not need internet; only the telemetry sender needs access to this Firebase endpoint.

POST https://test-c7bf3-default-rtdb.firebaseio.com/iperf3_streams.json

  "streamCode": "wifi_client_1",
  "interfaceName": "Wi-Fi",
