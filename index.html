Combined Throughput Plot

X-axis is numeric seconds from 0 to 60. The primary x value is elapsedEndSec from iperf. All streams use one shared global 0–60 second window, so one stream cannot replace or alternate with another.

Window: —
Smoothed solid lines = all stream throughput curves plotted together in the same global 0–60 s window with point markers hidden. No dashed reference lines are drawn, and the legend order is fixed by first stream appearance so it does not jump as new samples arrive.
